# Content Improvements Master Document

This document consolidates all recommended content-level improvements for the full toolset.

## Global content standards (apply to every tool)

1. Keep structure deterministic:
   - intro/context
   - objective/outcome
   - core content
   - assessment/check-for-understanding
   - next steps/extension
2. Enforce readability target by year group/reading age.
3. Require explicit examples and misconceptions where relevant.
4. Use actionable teacher language (what to do, when, and how to measure).
5. Preserve factual entities during rewrites; adapt tone and complexity only.

---

## Tool-by-tool content improvements

### Lesson Planner
- Enforce objective-success criteria alignment.
- Require at least 3 concrete teacher questions per phase.
- Include retrieval and formative checks in each lesson phase.

### Medium Term Planner
- Add week-by-week prerequisite checks.
- Add cumulative retrieval thread and progression checkpoints.
- Require measurable outcomes for each week.

### Quiz Generator
- Improve distractor quality (plausible but clearly incorrect).
- Require answer explanations and misconception notes.
- Balance question difficulty and cognitive demand.

### Rubric Generator
- Use observable verbs only.
- Ensure each level includes explicit evidence descriptors.
- Remove overlap between criteria.

### Comprehension Generator
- Force balanced question mix: literal/inferential/evaluative.
- Tag question reading demand.
- Include model answer stems.

### Exit Ticket
- Add confidence check question.
- Add misconception probe question.
- Keep closure prompts short and measurable.

### Flash Cards
- One concept per card.
- Avoid double-barrel definitions.
- Add quick retrieval cue per card.

### Vocabulary Builder
- Include student-friendly + technical definitions.
- Add bilingual equivalents when enabled.
- Add contextual example sentence.

### Behaviour Plan
- Convert advice into trigger-action-response format.
- Add review checkpoints and owners.
- Keep language specific and classroom-practical.

### Wellbeing Support
- Enforce supportive non-diagnostic phrasing.
- Include immediate safeguarding escalation options.
- Add clear boundaries for teacher role vs specialist referral.

### IEP Generator
- Every target must include baseline evidence.
- Add strategy, owner, review date, and success metric per target.
- Keep targets measurable and time-bound.

### Pupil Passport
- Strengths/interests first.
- Then barriers and practical adjustments.
- Include concise “what works / what to avoid” section.

### Smart Targets
- Reject vague verbs.
- Force measurable outcomes and cadence.
- Include evidence source to verify progress.

### Social Stories
- Keep first-person perspective.
- Use predictable sequence.
- Add explicit coping scripts.

### Report Comments
- Require evidence sentence + impact sentence + next-step sentence.
- Remove unsupported claims.
- Keep tone professional and balanced.

### Parent Newsletter
- Simplify jargon.
- Ensure clear actions, dates, and contacts.
- Add readability and tone checks before output.

### Text Rewriter
- Preserve named entities/facts.
- Only adjust complexity/tone/length requested.
- Highlight if any semantic risk detected.

### Presentation Maker
- Cap cognitive load per slide.
- One key idea + one check-for-understanding per slide.
- Enforce progression and recap prompts.

### Stories
- Ensure age-appropriate language and plot complexity.
- Add vocabulary reinforcement moments.
- Include comprehension checkpoint prompts.

### Differentiate
- Preserve original learning intent and marks demand.
- Adapt scaffolding depth by SEND profile.
- Keep structural parity across differentiated versions.

### SEND Screener
- Use neutral phrasing and confidence indicators.
- Include explicit “not a diagnosis” framing.
- Provide practical next-step signposting.

### Risk Assessment
- Use hazard-control-residual-risk structure.
- Assign accountable owners and review dates.
- Require concrete mitigation actions.

### Quiz Join
- Improve correctness feedback quality.
- Add spacing/revision prompts after responses.
- Add concise rationale for right/wrong answers.

---

## Priority order for implementation

1. High-stakes tools first (Wellbeing, IEP, SEND Screener, Risk, Report Comments).
2. Worksheet family next (Differentiate, Lesson Planner, Medium Term Planner, Comprehension, Exit Ticket, Vocabulary).
3. Engagement tools after (Stories, Flash Cards, Presentation, Quiz Join).

---

## Acceptance criteria

- Every tool output passes structure, readability, and clarity checks.
- High-stakes tools include safety language and escalation guidance.
- Assessment-oriented tools include evidence-backed answers/criteria.
- Differentiated outputs preserve intent while adapting support level.
